GITでのコマンド
・バージョン管理
　　管理したいディレクトリに移動後　git init ★
　　.gitフォルダが作成されるので管理開始
・GitHub側にリポジトリ作成
　　Webブラウザで作業する
・フロー
　　ファイル変更　　　　→　ステージングエリアへ追加　git add★
　　ステージングエリア　→　ローカルリポジトリへ　git commit★　　(まとめてコミットもできるが、1作業ごとにファイルを選んでコミットできる。まとめてなら.）
　　ローカルリポジトリ　→リモートリポジトリへ　　git push★
・コミット時にはMSG
　　1行目やったこと
　　2行目空行
　　3行目理由
　を書けばいい。
・ステータス確認　git status
